What does a hash function do?

Even a small variation in the input text generates a completely different fingerprint. This is what makes hashing so useful for detecting the slightest change to a file or protecting a password.

Why even Facebook can't see your password

When a user logs in, their password is not stored in plain text. Instead, it is transformed by a hash function into a unique, irreversible string of characters. This hash is what gets stored in Facebook’s servers—not the original password. Even Facebook administrators cannot reverse the hash to reveal the real password. This process protects users in case of data breaches and ensures that sensitive information remains secure.

Different hash = Different file

These two files may look identical, but their hashes tell a different story. A mismatch means the file has changed. It could be corrupted, tampered with, or even contain malware. Always verify a file’s hash to ensure its integrity and authenticity.
